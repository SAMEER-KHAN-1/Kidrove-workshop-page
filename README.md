# Kidrove Workshop Landing Page

A responsive workshop landing page built for the Kidrove internship assignment.

## Features

- Responsive React UI
- Workshop details section
- Learning outcomes
- FAQ section
- Registration form
- Express.js backend API
- Form validation

## Tech Stack

- React.js
- Tailwind CSS
- TypeScript
- Express.js
- Node.js

## Run Locally

### Frontend

cd client
npm install
npm run dev

### Backend

cd server
npm install
npm run dev
