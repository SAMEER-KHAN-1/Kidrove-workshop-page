# Kidrove Workshop Landing Page

Landing page for the **AI & Robotics Summer Workshop** by Kidrove — built as part of an internship assignment.

**Live Demo:** https://kidrove-workshop-page-six.vercel.app

---

## What's in it

- Hero section with workshop title, description, and enroll button
- Workshop details (age group, duration, mode, fee, start date)
- Learning outcomes
- Accordion FAQ section
- Registration form with client-side validation, loading states, and MongoDB storage
- Mobile-responsive layout with a hamburger navbar
- Smooth scroll navigation that works across all devices (including desktops with reduce-motion enabled)

## Tech Stack

**Frontend** — React, TypeScript, Tailwind CSS, Vite  
**Backend** — Express.js, TypeScript, Mongoose, MongoDB Atlas  
**Deployed on** — Vercel (frontend) + Render (backend)
