📝 Submission Note ~120 words

I approached this assignment by breaking the project into clear frontend and backend layers. On the frontend, I used React with Vite and Tailwind CSS v4 — the component structure maps directly to the page sections (Hero, WorkshopInfo, LearningOutcomes, FAQSection, RegistrationForm), keeping each file focused and readable. All workshop data is extracted into a single workshopData.js file so it's easy to update.

Design decisions: I went with a dark navy hero section to create an immediate "tech" impression, with orange accents for energy and CTAs. The form includes real-time client-side validation and a simulated async submit flow that mirrors the actual API call.

Future improvements I'd add: persistent form data with React Query, email confirmation via Nodemailer, an admin dashboard to view registrations, and MongoDB for persistent storage instead of the in-memory array.
